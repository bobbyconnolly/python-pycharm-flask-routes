Library Project Setup Guide

1. Project Structure

Create this folder structure in PyCharm:

library_system/ ├── app.py # Main Flask application ├── schema.sql # Database schema ├── init_db.py # Database initialization script ├── requirements.txt # Project dependencies ├── static/ │ └── css/ │ └── style.css └── templates/ └── index.html # Main template

2. Dependencies

Create requirements.txt with:

Flask==3.0.0
Run pip install -r requirements.txt in your PyCharm terminal to install dependencies.

3. Database Schema

Create schema.sql:

-- Drop tables if they exist
DROP TABLE IF EXISTS books;
DROP TABLE IF EXISTS loans;

-- Create books table
CREATE TABLE books (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    title TEXT NOT NULL,
    isbn TEXT UNIQUE,
    author TEXT NOT NULL,
    status TEXT DEFAULT 'available'
);

-- Create loans table
CREATE TABLE loans (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    book_id INTEGER NOT NULL,
    member_id INTEGER NOT NULL,
    loan_date TEXT NOT NULL,
    due_date TEXT NOT NULL,
    returned_date TEXT,
    FOREIGN KEY (book_id) REFERENCES books (id)
);

4. Database Initialization

Create init_db.py:

import sqlite3

def init_db():
    # Connect to database
    conn = sqlite3.connect('library.db')
    
    # Read schema file
    with open('schema.sql', 'r') as f:
        schema = f.read()
    
    # Execute schema
    conn.executescript(schema)
    
    # Add some test data
    conn.executescript("""
        INSERT INTO books (title, isbn, author) VALUES 
        ('Python Basics', '1234567890123', 'John Doe'),
        ('Flask Web Dev', '9876543210123', 'Jane Smith');
    """)
    
    conn.commit()
    conn.close()
    print("Database initialized!")

if __name__ == '__main__':
    init_db()

7. Running the Project

Create app.py (where you'll implement your routes) with this basic structure:

from flask import Flask, request, render_template
import sqlite3
from datetime import datetime, timedelta

app = Flask(__name__)

def get_db():
    conn = sqlite3.connect('library.db')
    conn.row_factory = sqlite3.Row
    return conn

@app.route('/')
def index():
    return render_template('index.html')

# Your route implementations will go here

if __name__ == '__main__':
    app.run(debug=True)

Setup Steps

  1. Create project folder structure
  2. Copy all files with their contents
  3. Run pip install -r requirements.txt
  4. Run python init_db.py to initialize the database
  5. Run python app.py to start the server
  6. Visit http://localhost:5000 in your browser
